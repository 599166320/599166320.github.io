3.docker常用的简单管理命令

2017-10-02
  • docker

查看本地docker镜像

docker images

在docker容器中执行linux命令

#如果本地没有想过镜像，会自动下载
[root@bogon ~]# docker run centos:centos6 uname -a
Unable to find image 'centos:centos6' locally
centos6: Pulling from library/centos

后台运行docker容器

docker run -dit -P nginx

查看运行的docker

docker ps
#可以看到CONTAINER_ID

查看docker的日志

docker logs CONTAINER_ID

登录到docker容器

docker exec -it CONTAINER_ID

停止容器

docker stop CONTAINER_ID

查看帮助

docker run --help

删除容器

docker rm CONTAINER_ID

删除镜像

docker rmi IMAGE_ID

#注意IMAGE_ID可以使用docker images来看到
